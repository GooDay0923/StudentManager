# StudentManager

## 项目介绍

　　基于Spring+SpringMVC+Mybatis开发系统架构

----|------|----

## 环境搭建

#### 开发工具:
- MySql: 数据库
- Tomcat: 应用服务器
- Git: 版本管理
- IntelliJ IDEA: 开发IDE
- Navicat for MySQL: 数据库客户端

#### 开发环境：
- Jdk8
- Mysql5.6
- Redis
